HOW TO PLAY Guess the Color Game(GCG):

  1. There are three colors involved: red, green and blue.
  2. To start the game, click the "START" button.
  3. When color has been typed into the space provided, the SAVE button should be clicked or ENTER button on the keyboard and the right color will be displayed as background color of the output box.
  4. When either player types in the color in the space provided, the color should be typed in lowercase. E.g. If Player 1 color guess is red, he/she should type in "red".
  5. For every color guessed corretly, 5 points are earned and for every color guessed incorrectly, 2 points are deducted
  6. The first palyer must guess a color before the second player can proceed to guess a color.
  7. Each player has ten trails before the winner is declared based on the highest point.

It should be noted that, if a color is misspelled or if no color is input and the SAVE button is clicked, it would be assumed that, the wrong color is guessed.
